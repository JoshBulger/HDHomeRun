Hdhomerun

Simple bash script to scan your HDHomeRun and create strm files for use with XBMC (intended for use with Linux)

Download this project as a .zip file Download this project as a tar.gz file

Once you have installed hdhomerun_config, instructions here:Linux HDHomeRun installation guide, download the hdhomerun.sh script and run it. You may have to add execute permissions with chmod +x. Use tar -zxvf {file.tar.gz} To extract the script. On the rare occasion it may require to be run twice (if nothing shows up in your Live TV folder the first time).

Afterwards add ~/Videos/Live TV to your video file in XBMC.

You should now be able to stream individual channels via XBMC by opening Videos>Live TV>'Your Channel'
