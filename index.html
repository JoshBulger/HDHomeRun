Hdhomerun

Simple bash script to scan your HDHomeRun and create strm files for use with XBMC (intended for use with Linux)

Download Download View Hdhomerun on GitHub

Info:

This project takes advantage of hdhomerun_config and then parses the scanned data from tuner1 into stream files to be used with XBMC. This script is intended to only work with a single device. As always, use code found on the internet at your own risk. That being said you are more than welcome to download and view the code using your favorite editor.

Installation and use:

Once you have installed hdhomerun_config, instructions here:Linux HDHomeRun installation guide, continue on with the below instructions.

Change to the directory downloaded to:
$ cd ~/Downloads/

Extract the tar.gz
$ tar -zxvf joshbulger-HDHomeRun-dfdc052.tar.gz

Go into the extracted directory
$ cd joshbulger-HDHomeRun-dfdc052

Run the script.
$ ./homeruntv.sh

Check to make sure that you have *.strm files
$ ls ~/Videos/Live\ TV

On rare occasions it may require to be run twice (if nothing shows up in your Live TV folder the first time).

Afterwards add ~/Videos/Live TV to your video file in XBMC.

You should now be able to stream individual channels via XBMC by opening Videos>Live TV>'Your Channel'

*note that you are required to have XBMC installed to view the streams within XBMC.
